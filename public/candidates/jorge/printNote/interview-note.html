🎯 实现反函数 / Implement Inverse Function

Given a black-box function f(x) that is strictly monotonically increasing (non-negative integers → non-negative integers), implement its inverse g(y): given y, find x such that f(x) = y. A valid x always exists.
给定严格单调递增的黑盒函数 f(x)（非负整数→非负整数），实现反函数 g(y)：给定 y，找到 x 使得 f(x) = y。答案一定存在。
期望解法：Binary Search · O(log n) · 关键：确定搜索上界
