🏢 "你们组是做什么的？" / "What does your team work on?"

🎤 回答要点 / Answer Hints

Step 1: 介绍 Fabric / Introduce Fabric

"我们属于 Microsoft Fabric。Fabric 是微软的一站式数据分析平台 (SaaS)，把数据工程、数据科学、实时分析、数据仓库和 BI 报表统一在一个平台上，底层共享一个叫 OneLake 的统一数据湖。"
"We are part of Microsoft Fabric. Fabric is Microsoft's unified SaaS analytics platform that brings together data engineering, data science, real-time analytics, data warehousing, and BI reporting on a single platform, with a shared data lake called OneLake underneath."

Step 2: 介绍 Data Science & Data Engineering / Introduce DS & DE

"在 Fabric 下面，我们团队负责两个产品：Data Science 和 Data Engineering。"
"Under Fabric, our team owns two products: Data Science and Data Engineering."

📊 Data Science

"Data Science 是一个机器学习平台，让用户可以在 Fabric 里完成从数据探索、模型训练到部署上线的完整 ML 工作流。它集成了 MLflow 做实验追踪和模型注册，整体体验比较像 Databricks。"
"Data Science is a machine learning platform that lets users complete the full ML workflow inside Fabric — from data exploration and model training to deployment. It integrates MLflow for experiment tracking and model registry. The overall experience is similar to Databricks."

⚙️ Data Engineering

"Data Engineering 提供基于 Apache Spark 的大规模数据处理能力，核心体验是 Notebook（在线代码编辑器）和 Lakehouse（数据湖仓）。用户可以用 Python/PySpark/Scala 写数据转换和分析任务。"
"Data Engineering provides large-scale data processing powered by Apache Spark. The core experiences are Notebook (an online code editor) and Lakehouse (a lake-house data store). Users write data transformation and analysis jobs in Python / PySpark / Scala."

💻 "UX 岗位具体做什么？" / "What does the UX role actually do?"

🎤 回答要点 / Answer Hints
"我们负责的是 Data Science 和 Data Engineering 这两个产品的前端界面开发。"
"We own the frontend UI development for both the Data Science and Data Engineering products."

📊 Data Science 前端

"Data Science 这边我们用的是 MLflow 开源项目，整个体验会比较像 Databricks——实验管理界面、模型注册表、运行对比等。"
"On the Data Science side, we build on the MLflow open-source project. The experience is similar to Databricks — experiment management UI, model registry, run comparison, etc."

⚙️ Data Engineering 前端

"Data Engineering 这边主要做的是 Python Notebook 在线编辑器，体验比较像 Google Colab——在浏览器里写代码、运行 Spark 任务、看输出结果。"
"On the Data Engineering side, the main thing we build is the Python Notebook online editor. The experience is similar to Google Colab — writing code in the browser, running Spark jobs, and viewing output."
